# todolist
### Demo
https://misty1636.github.io/Todolist/dist/

+ 在My Tasks頁面可以拖拽各事項
+ 排序以 星號 > 托拽順序，星號必為最上面
+ 可對每個事項再做編輯，且編輯中可以點擊星號
+ 使用Vue Cli 3、Vue.Draggable、LocalStorage、Bootstrap、Scss、Eslint(Airbnb)
